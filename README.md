# AlphaBetaLabs

## What This Repo Is
- An authority-ordered corpus for building AlphaBetaLabs systems: Governance -> Canon -> Enforcement -> Manifest -> Engines/Plugins/Systems -> Research -> Experiments -> Archive.

## What This Repo Is NOT
- Not an implementation-first codebase.
- Not a place to redefine Canon from below.
- Not a dumping ground for temporary structures.

## Identity And Immutability (Locked)
- `AlphaBeta Labs` (the lab/creator/brand) is mutable: humans and research evolve.
- Immutability starts at the artifact boundary: engine constitutions (per version), frozen semantic manifests, and signed canonical outputs.
- Drift is allowed in research and mirrors/bridges; drift is forbidden inside a released engine constitution version (changes require a new version via governance).
- `AlphaBetaLabs.io` is a distribution surface for stabilized engines/products (outside this repoâ€™s authority layers).
